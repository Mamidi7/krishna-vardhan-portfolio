Teaching kids to code with animated HTML worksheets

Zero setup, zero install — just a browser. I built animated HTML/SVG worksheets for 6-18 year olds using shadcn-inspired design, CSS animations, and cubic-bezier(0.16,1,0.3,1) easing.
